# Fernando's Voyage - Luxury Travel Portal

**Fernando's Voyage** es una plataforma premium para descubrir y explorar destinos de viaje alrededor del mundo de manera fácil y eficiente.

## Personalización
Este proyecto ha sido desarrollado y personalizado por **Fernando Farfán**.

## Funcionalidades

- **Búsqueda de Destinos:** Permite a los usuarios buscar destinos por palabras clave como "playa", "templo" o nombres de países y ciudades.
- **Visualización de Recomendaciones:** Muestra recomendaciones de destinos basadas en la búsqueda del usuario con una interfaz moderna y fluida.
- **Interacción con Usuarios:** Permite a los usuarios contactar a **Fernando Farfán** a través de un formulario de contacto profesional.
